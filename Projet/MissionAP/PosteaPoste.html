Réseau Poste à Poste

Contexte de la Mission

Configuration d'un réseau local simple permettant la communication et le partage de ressources entre plusieurs postes de travail Windows.

Cette mission s'inscrit dans l'apprentissage des bases du réseautage et de la configuration des systèmes Windows en environnement de groupe de travail.

Configuration Réalisée

Configuration IP : Paramétrage des adresses IP statiques sur le même sous-réseau pour permettre la communication entre les postes. Adresses IP : 192.168.1.x, Masque de sous-réseau : 255.255.255.0, Pas de passerelle (réseau local uniquement).

Partages de Fichiers : Configuration des partages de dossiers avec gestion des permissions d'accès pour chaque utilisateur. Dossiers partagés par poste, permissions de lecture/écriture et accès par nom d'utilisateur.

Services Windows : Activation des services nécessaires pour le fonctionnement du réseau et des partages. Service de partage de fichiers, découverte du réseau et gestionnaire d'ordinateurs.

Tests de Connectivité

Ping : Tests de ping entre tous les postes pour vérifier la connectivité de base.

Accès aux Partages : Vérification de l'accès aux dossiers partagés depuis chaque poste.

Authentification : Tests de connexion avec différents comptes utilisateurs.

Performance : Mesure des vitesses de transfert de fichiers entre postes.

Retour d'Expérience

Cette mission m'a permis de comprendre les bases du réseautage et l'importance de la configuration correcte des paramètres réseau pour assurer la communication entre postes. Bien que simple, cette configuration m'a donné les fondements nécessaires pour comprendre les réseaux plus complexes et l'importance de la documentation des configurations.
