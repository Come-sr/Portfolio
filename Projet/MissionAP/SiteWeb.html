Conception et realisation d'un site web

Informations Complémentaires
Outils : Visual Studio, Github
Période : 15/11 - 20/12
Compétences Mises en Oeuvre
- Développer la présence en ligne de l’organisation

- Travailler en mode projet

- Mettre à disposition des utilisateurs un service informatique

Contexte :

Dans ce projet, Je suis chargee de developper un site web J'ai choisi de faire une page web pour un bar de la ville, le gaulois. le site est disponible a l'adresse : https://24kms.github.io/gimenez/

Pour la création de ce site j'ai utilisé mes competences en HTML et CSS

Taches :

1) Développement d’un site web « vitrine » autour d’un centre d’intérêt et/ou d’une entreprise ou organisation réelle ou fictive.

2) Etude et choix d'hébergeur.

3) Hébergement et référencement du site Web.

4) Présentation orale du travail effectué.
