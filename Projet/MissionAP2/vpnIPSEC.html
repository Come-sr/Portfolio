VPN IPSec Site-à-Site

IPSec Cisco VPN Packet Tracer
Objectif : Déployer un tunnel VPN IPSec entre deux routeurs Cisco pour sécuriser les communications inter-sites.

Contexte

Dans le cadre de notre projet de mise en place d'infrastructure réseau, nous avons déployé un VPN IPSec sur deux routeurs Cisco pour sécuriser les communications entre deux sites.

Qu'est-ce qu'un VPN IPSec ?

Un VPN IPsec (Internet Protocol Security) est une technologie utilisée pour sécuriser les communications sur un réseau IP. Il chiffre les données échangées, garantissant leur confidentialité et leur intégrité, ce qui est essentiel pour connecter des sites distants de manière sécurisée.

Schéma d'un tunnel VPN IPSec site-à-site

Configuration et Tests

Nous avons d'abord configuré les équipements sur Packet Tracer pour nous familiariser avec les commandes. Nous avons ensuite mis en place le VPN IPSec sur deux routeurs Cisco physiques et effectué les tests nécessaires pour assurer la bonne communication à travers le tunnel VPN.

Ce que j'ai appris

Ce projet m'a permis de découvrir les VPN et de comprendre leur fonctionnement pratique. J'ai appris à configurer un tunnel IPSec sur des routeurs Cisco et à effectuer les vérifications nécessaires pour valider la connexion sécurisée.
