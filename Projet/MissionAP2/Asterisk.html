Administration Réseaux - Mise en Place de la Téléphonie avec Asterisk

Contexte

Dans le cadre de notre projet de mise en place de services de téléphonie IP, nous avons déployé un serveur Fedora avec le service Asterisk pour la gestion des appels, avec des fichiers de configuration pjsip.conf et extensions.conf. Nous avons également configuré les services DHCP et TFTP pour la prise en charge des téléphones IP, et installé le logiciel 3CXPhone sur les postes sous Windows 10.

1. Schéma Réseau

Notre infrastructure réseau inclut un serveur Asterisk, Deux téléphones IP virtuel sur des machines Windows 10. Cette configuration permet une communication centralisée et une gestion efficace des ressources téléphoniques au sein de l'organisation.

2. Configuration Asterisk

Nous avons configuré les fichiers pjsip.conf pour la gestion des utilisateurs et extensions.conf pour définir les extensions et les règles de routage des appels. Ces fichiers permettent de gérer les appels et d’assurer la bonne communication entre les différents postes.


; Exemple de configuration dans pjsip.conf
[6001]
type = endpoint
context = local
disallow = all
allow = ulaw
auth = auth6001
aors = 6001
