Cisco Command Cheatsheet

Basic Configuration

Command Description
enable Enter privileged EXEC mode.
disable Return to user EXEC mode.
configure terminal Enter global configuration mode.
hostname [name] Set the hostname of the device.
copy running-config startup-config Save the current configuration.
show running-config Display the current configuration.
show version Display system hardware and software status.
reload Reboot the router or switch.
clock set HH:MM:SS DD MMM YYYY Set the system clock.

VLANs and Trunking

Command Description
vlan [id] Create a VLAN.
name [name] Name the VLAN.
interface vlan [id] Create a VLAN interface.
switchport mode access Set the port to access mode.
switchport access vlan [id] Assign the port to a VLAN.
switchport mode trunk Set the port to trunk mode.
switchport trunk allowed vlan [list] Configure allowed VLANs on a trunk port.
show vlan brief Display a summary of configured VLANs.

Routing and IP Configuration

Command Description
interface [interface] Enter configuration mode for the specified interface.
ip address [ip] [mask] Configure an IP address on the interface.
no shutdown Activate the interface.
shutdown Deactivate the interface.
ip route [network] [mask] [next-hop] Add a static route.
show ip route Display the IP routing table.

OSPF Routing

Command Description
router ospf [process-id] Enable OSPF and enter router configuration mode.
network [ip] [wildcard-mask] area [id] Specify the networks to be advertised in OSPF.
show protocols Display the configured routing protocols.

VPN Configuration

Command Description
crypto isakmp policy [priority] Create an ISAKMP policy.
authentication pre-share Specify pre-shared key authentication.
crypto isakmp key [key] address [ip] Configure the ISAKMP pre-shared key.
crypto ipsec transform-set [name] Define an IPsec transform set.
crypto map [name] [seq] ipsec-isakmp Create a crypto map.
set peer [ip] Specify the peer IP address.
set transform-set [name] Apply the transform set to the crypto map.
interface [interface] Enter interface configuration mode.
crypto map [name] Apply the crypto map to the interface.

Useful Linux Commands

Command Description
sudo apt update && sudo apt upgrade Update and upgrade packages (Debian/Ubuntu).
sudo dnf update && sudo dnf upgrade Update and upgrade packages (Fedora).
sudo apt install [package] Install a package (Debian/Ubuntu).
sudo dnf install [package] Install a package (Fedora).
ip a Display network interfaces.
ping [ip/hostname] Test network connectivity.
systemctl status [service] Check the status of a service.
nano /etc/network/interfaces Edit network interface configuration (Debian/Ubuntu).
sudo reboot Reboot the system.
