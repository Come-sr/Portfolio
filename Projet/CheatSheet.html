Cheatsheet des Commandes Cisco

Configuration de Base

Commande Description
enable Accéder au mode EXEC privilégié.
disable Retourner au mode EXEC utilisateur.
configure terminal Passer en mode configuration globale.
hostname [nom] Définir le nom de l’appareil.
copy running-config startup-config Sauvegarder la configuration actuelle.
show running-config Afficher la configuration actuelle.
show version Afficher les informations sur le matériel et logiciel système.
reload Redémarrer le routeur ou le switch.
clock set HH:MM:SS JJ MMM AAAA Régler l’horloge système.

Configuration VLAN et Trunk

Commande Description
vlan [id] Créer un VLAN.
name [nom] Nommer le VLAN.
interface vlan [id] Créer une interface VLAN.
switchport mode access Définir le port en mode accès.
switchport access vlan [id] Assigner le port à un VLAN.
switchport mode trunk Définir le port en mode trunk.
switchport trunk allowed vlan [liste] Configurer les VLAN autorisés sur un port trunk.
show vlan brief Afficher un résumé des VLAN configurés.

Routage et Configuration IP

Commande Description
interface [interface] Accéder au mode configuration pour l’interface spécifiée.
ip address [ip] [masque] Configurer une adresse IP sur l’interface.
no shutdown Activer l’interface.
shutdown Désactiver l’interface.
ip route [réseau] [masque] [next-hop] Ajouter une route statique.
show ip route Afficher la table de routage IP.

Commandes de Sécurité

Commande Description
password {valeur-pass} Définit un mot de passe pour le login.
enable secret {valeur-pass} Définit un mot de passe pour le mode privilégié.
service password-encryption Chiffre les mots de passe dans la configuration.
crypto key generate rsa Génère les clés nécessaires pour SSH.
ip access-list [standard | extended] [nom] Définir une liste de contrôle d'accès IP.
permit / deny source [masque] Permet ou bloque les paquets dans une ACL.
switchport port-security Activer la sécurité de port sur l’interface.

Configuration OSPF

Commande Description
router ospf [id] Activer OSPF et entrer en mode configuration.
network [ip] [masque joker] area [id] Spécifier les réseaux OSPF.
show ip ospf neighbor Afficher les voisins OSPF.

Commandes de Dépannage

Commande Description
ping [ip/hostname] Vérifie la connectivité réseau.
show ip interface brief Afficher l’état des interfaces IP.
show interfaces Afficher des informations détaillées sur les interfaces.
show cdp neighbors Voir les voisins CDP (Cisco Discovery Protocol).

2. Configuration de base du Switch Cisco

Pour configurer un switch Cisco en mode de base :

Switch> enable
Switch# configure terminal
Switch(config)# hostname MonSwitch
Switch(config)# interface vlan 1
Switch(config-if)# ip address 192.168.1.1 255.255.255.0
Switch(config-if)# no shutdown
Switch(config-if)# exit
Switch(config)# ip default-gateway 192.168.1.254
Switch(config)# exit
Switch# copy running-config startup-config

3. Configuration de base du Routeur Cisco

Pour configurer un routeur Cisco avec une adresse IP de base :

Router> enable
Router# configure terminal
Router(config)# hostname MonRouteur
Router(config)# interface gigabitethernet 0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# ip routing
Router(config)# exit
Router# copy running-config startup-config

4. Configuration des VLANs (Switch Cisco)

Les VLANs (Virtual Local Area Networks) sont utilisés pour segmenter le réseau en sous-réseaux logiques. Voici comment configurer un VLAN sur un switch Cisco :

Switch(config)# vlan 10
Switch(config-vlan)# name "VLAN-10"
Switch(config-vlan)# exit
Switch(config)# interface range fastethernet 0/1 - 0/24
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 10
Switch(config-if-range)# exit

5. Configuration d'un Routeur avec plusieurs interfaces

Un routeur Cisco peut être configuré avec plusieurs interfaces pour séparer des réseaux. Voici un exemple pour configurer deux interfaces avec des adresses IP distinctes :

Router> enable
Router# configure terminal
Router(config)# interface gigabitethernet 0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# interface gigabitethernet 0/1
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit

6. Vérification de la configuration

Utilisez les commandes suivantes pour vérifier l'état et la configuration des interfaces réseau :

Router# show running-config
Router# show ip interface brief
Router# show ip route
Switch# show vlan brief

7. Sauvegarder la configuration

Pour enregistrer la configuration dans la mémoire de démarrage et la conserver après un redémarrage, utilisez la commande suivante :

Router# copy running-config startup-config
Switch# copy running-config startup-config
